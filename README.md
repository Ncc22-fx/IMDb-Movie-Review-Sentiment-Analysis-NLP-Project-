# Building Model on IMDb-Movie-Review-Sentiment-Analysis-NLP-Project-

## Project Overview
This project focuses on using Natural Language Processing (NLP) and machine learning techniques to classify IMDb movie reviews as either positive or negative based on textual sentiment.
The goal was to build a sentiment classification model capable of understanding customer opinions from unstructured text data and transforming those reviews into actionable insights.
This project demonstrates practical applications of NLP in customer feedback analysis, brand monitoring, review systems, and recommendation platforms.

## Problem Statement

Online platforms receive massive volumes of user-generated reviews every day.
Manually reading and interpreting thousands of reviews is inefficient and time-consuming.

Businesses need automated systems that can:

Detect customer sentiment
Understand public perception
Monitor brand reputation
Improve recommendation systems

This project solves that challenge by automatically classifying review sentiment using NLP techniques.

### Dataset
The dataset used was the IMDb Movie Reviews Dataset, containing:
Movie review text
Sentiment labels:
Positive
Negative

Project Workflow
1. Text Data Cleaning

Preprocessed raw text by:

Removing punctuation
Removing special characters
Converting text to lowercase
Removing stopwords
Tokenization
Stemming/Lemmatization
Feature Engineering

Converted text into numerical format using:

Bag of Words
TF-IDF Vectorization

###Machine Learning Models and Deep Leanring Models Used
Tested multiple classification algorithms including:

Logistic Regression
Naive Bayes
Support Vector Machine (SVM)
Convolutional Neural Network (CNN)
LSTM
GRU
## Model Evaluation

Performance was evaluated using:

Accuracy
Precision

Confusion Matrix

The final model successfully classified movie review sentiment with strong predictive performance.
